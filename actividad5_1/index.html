Funcionamiento de las etiquetas HTML

  • <dl >

    Representa una lista descriptiva, y esta se utiliza para representar la lista de descripción.

  • <dt >

    • Especifica un término en una descripción o lista de definiciones.

  • <dd >

    • Provee detalles acerca de o la definición de un término precedente en una lista de descripciones.

    Ejemplo:

    <dl>
    <dt>¿Que abra mas alla del imenzo mar? </dt>
    <dd>Seguramente abra un imenzo vacio</dd>
    </dl>

    ¿Que abra mas alla del imenzo mar?
    Seguramente abra un imenzo vacio

  • < blockquote >

    Se utiliza para mostrar las citas largas

    Ejemplo:

    <blockquote> cite="https://www.netflix.com/mx/title/60000171?source=35">
    <p>Dos pícaros españoles se topan con un mapa del tesoro de El Dorado y se embarcan como polizones en un barco. Cuando encuentra la ciudad, el sacerdote principal los proclama dioses dentro de un elaborado plan para tomar el control de la ciudad.</p>
    </blockquote>

    Dos pícaros españoles se topan con un mapa del tesoro de El Dorado y se embarcan como polizones en un barco. Cuando encuentra la ciudad, el sacerdote principal los proclama dioses dentro de un elaborado plan para tomar el control de la ciudad.

  • < Q >

    Define una cita corta

    Ejemplo:

    <p<q>En el siglo XVI, dos estafadores españoles ganan un mapa hacia la legendaria ciudad del oro</q>
    Pero, al llegar a El Dorado, las cosas no suceden de acuerdo con el plan. </p>

    En el siglo XVI, dos estafadores españoles ganan un mapa hacia la legendaria ciudad del oro
    Pero, al llegar a El Dorado, las cosas no suceden de acuerdo con el plan.

  • < cite >

    Define el titulo de una obra

    Ejemplo:

    <cite>Ejemplo de una etiqueta cite </cite>
    Ejemplo de una etiqueta cite

  • < abbr >

    Define la abreviatura o forma corta de un elemento.

    Ejemplo:

    <abbr> title="El camino al dorado">ECAD</abbr>
    ECAD

  • < sub >

    Sirve para agregar un texto de subíndice al documento HTML.

  • < sup >

    Sirve para definir el texto del superíndice.

    Ejemplo:

    <p>Prueba <sub>Subindice </sub> </p>

    Prueba Subindice

    <p>Prueba <sup>Superindice </sup> </p>

    Pruebasuperíndice

  • < code >

    se ocupa para definir el fragmento del código informático.

    Ejemplo:

    < code >
    #include < stdio.h >
    int main() {
    printf("Hola mundo");
    }
    < /code > 
    
  #include
  int main() {
      printf("Hola mundo");
  }

  • < pre >

    se ocupa para definir el bloque de texto preformateado.

    Ejemplo:

    < pre > El camino hacia El Dorado
    Dos pícaros españoles se topan con un mapa del tesoro. < /pre > 
    El camino hacia El Dorado 
    
Dos pícaros españoles se topan con un mapa del tesoro.

  • < var >

    Es una etiqueta de frase y esta se utiliza para especificar la variable en una ecuación matemática.

    Ejemplo:

    < var > Ejemplo de variable 2+2 < /var >
    Ejemplo de variable 2+2

  • < kbd >

    Es una etiqueta de frase y esta se utiliza para definir la entrada del teclado.

    Ejemplo:

    < p>
    Comando para pegar un
    < kbd > Ctrl < /kbd > + < kbd > C < /kbd >
    < /p >

    Comando para pegar un texto Ctrl + C

  • < samp >

    Es una etiqueta de frase y se utiliza para definir el texto de salida.

    Ejemplo:

    < samp > Ejemplo de la etiqueta samp </samp >
    Ejemplo de la etiqueta samp

  • < time >

    Se utiliza para mostrar la fecha/hora legible por humanos.

    Ejemplo:

    < p > Abrimos todos los dias de las < time > 11:00 < /time > a las < time > 19:00 < /time > todos los dias < /p >

    Abrimos todos los dias de las 11:00 a las 19:00 todos los dias
